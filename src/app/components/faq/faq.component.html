Preguntas Frecuentes

¿Son accesibles estos componentes?

Sí, estos componentes están diseñados siguiendo las mejores prácticas de accesibilidad, aunque siempre deberás revisar el contraste de colores en tu implementación final.

¿Por qué no un paquete NPM?

Preferimos el enfoque de copiar y pegar para que tengas control total sobre el código y puedas personalizarlo sin restricciones según tus necesidades.

¿Cómo puedo contribuir?

Puedes contribuir a este proyecto a través de GitHub mediante pull requests. Todas las ideas y mejoras son bienvenidas para hacer crecer esta biblioteca.

¿Funciona con Angular Material?

Esta biblioteca está pensada para ser independiente, pero puedes integrarla con Angular Material. Sin embargo, ten en cuenta que el estilo neobrutalista puede entrar en conflicto con el diseño Material.
